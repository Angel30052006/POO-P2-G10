## 📖 Descripción

Proyecto académico desarrollado para la materia de Programación Orientada a Objetos.

El sistema permite gestionar una liga deportiva mediante consola, administrando torneos, equipos, jugadores, entrenadores, árbitros y partidos. Además, permite registrar resultados, consultar estadísticas y actualizar automáticamente la tabla de posiciones.

## ✨ Funcionalidades Principales

### 🏆 Gestión de Torneos
- Crear torneos.
- Administrar equipos participantes.
- Consultar información de torneos.

### 👥 Gestión de Equipos
- Registrar equipos.
- Consultar equipos registrados.
- Administrar jugadores y entrenador asignado.

### 🏃 Gestión de Jugadores
- Registrar jugadores.
- Asignar jugadores a equipos.
- Consultar estadísticas individuales.

### 🎯 Gestión de Entrenadores
- Registrar entrenadores.
- Asignar entrenadores a equipos.

### 🧑‍⚖️ Gestión de Árbitros
- Registrar árbitros.
- Asignar árbitros a partidos.

### ⚽ Gestión de Partidos
- Programar partidos.
- Consultar calendario de encuentros.
- Registrar resultados.

### 📊 Consultas y Reportes
- Tabla de posiciones automática.
- Máximo goleador.
- Equipo con más victorias.
- Historial de partidos.
- Estadísticas generales.

## ▶️ Ejecución

1. Clonar el repositorio.
2. Abrir el proyecto en NetBeans, IntelliJ IDEA o Visual Studio Code.
3. Compilar utilizando Java 17 o superior.
4. Ejecutar la clase `Main`.
5. Utilizar el menú principal para acceder a las funcionalidades del sistema.
